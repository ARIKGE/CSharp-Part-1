# Hello World

## Description
Create, compile and run a **Hello World** application - it should print the string `Hello, C#!` on the console. Submit the code at the contest in [www.bgcoder.com](http://bgcoder.com/Contests/314/CSharp-Fundamentals-01-Introduction-to-Programming).

- There is no input for this task.

## Output
- Print `Hello, C#!` on the console.

- Memory limit: **8MB**
